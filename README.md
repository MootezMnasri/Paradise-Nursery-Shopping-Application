# Paradise Nursery - Shopping Cart Application

An online plant shop offering a variety of house plants with a fully functional shopping cart experience.

## Project Overview

Paradise Nursery is an e-commerce web application where users can browse and purchase a curated selection of houseplants. The app features a clean, responsive design and a seamless shopping experience.

## Features

- **Landing Page** – Welcome page with a hero section and a call-to-action linking to the product listing.
- **Product Listing** – Browse plants across multiple categories with images, descriptions, and prices.
- **Shopping Cart** – Add/remove items, adjust quantities, view subtotals, and checkout.
- **Persistent Cart** – Cart state is saved in localStorage across page visits.
- **Responsive Design** – Works on desktop, tablet, and mobile devices.

## Plant Categories

- **Aromatic Plants** – Lavender, Rosemary, Mint, Basil, Thyme, Lemongrass
- **Medicinal Plants** – Aloe Vera, Echinacea, Chamomile, Turmeric, Ginger, Ashwagandha
- **Air Purifying Plants** – Snake Plant, Spider Plant, Peace Lily, Pothos, Rubber Plant, Boston Fern

- CSS3
- Vanilla JavaScript
- GitHub Pages (Deployment)
